
# HTBX by 3A1W
![HTBX Title](https://i.imgur.com/s8r4zn7.png "HTBX")

A bullet hell game created in Java with Eclipse IDE.

# Controls
![HTBX Controls](https://i.imgur.com/yBkqjDE.gif "Controls")

W - Move forward.
<br /> A - Bank left.
<br /> S - Move backward.
<br /> D - Bank right.
<br /> Left Mouse Button (LMB) - Shoot at cursor.
